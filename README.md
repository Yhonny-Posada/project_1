#### Desctription:
#### PetCare is a mobile application built with React Native CLI that allows users to manage pet information. Users can register new pets through a multi-step form, browse the list of registered pets, view detailed profiles for each one, and access an auto-rotating care tips screen. The app demonstrates core React Native concepts including Stack and Tab Navigation, parameter passing between screens, and the useState / useEffect hooks.
